## Development

### Available Scripts

- `bun run build` - Build the Nuxt application
- `bun run dev` - Start the Nuxt development server
- `bun run generate` - Generate static site
- `bun run preview` - Preview the built application

## Features

- WebContainer management interface
- Container creation and deletion
- Command execution within containers
- Package installation and script running
- File listing and port mapping
