# tracing-ui

Application for tracing-ui

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```

## Development

```bash
bun run build
bun run test
```

## Available Scripts

- `dev`: nuxt dev
- `build`: nuxt build
- `preview`: nuxt preview
- `format`: dprint fmt
- `lint`: tsc --noEmit && oxlint --fix --type-aware
- `test`: vitest
- `postinstall`: nuxt prepare
- `watch`: bun --watch src/index.ts
