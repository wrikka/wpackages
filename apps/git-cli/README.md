# git-cli

An interactive Git CLI tool built with TypeScript and @clack/prompts.

## Features

- Interactive prompts for common Git commands.
- Simplifies complex Git operations.
- Built with modern tools for a great developer experience.

## Installation

Install the CLI globally using your preferred package manager:

```bash
bun add -g git-cli
```

## Usage

Run the CLI using the `git-cli` command:

```bash
git-cli
```

This will open an interactive menu to guide you through various Git commands.
