# Node Modules Analyzer

A CLI tool to analyze `node_modules` directory and provide insights.

## Features

*   **Interactive UI**: A beautiful and intuitive terminal interface.
*   **Size Analysis**: Visualize disk usage by package.
*   **Duplicate Finder**: Identify and list duplicate dependencies.
*   **Why is it here?**: Trace why a specific package is included.
*   **License Audit**: List all licenses used in the project.
*   **Vulnerability Scan (coming soon)**: Check for known security vulnerabilities.

## Usage

```bash
bun start <path-to-project>
```
