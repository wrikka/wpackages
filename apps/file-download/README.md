# My CLI

A CLI tool for development tasks.

## Installation

```bash
bun install
```

## Usage

To start the CLI tool in interactive mode:

```bash
bun dev
```

## Available Scripts

- `bun dev`: Run the CLI in watch mode.
- `bun build`: Build the project for production.
- `bun lint`: Lint and format the code.
- `bun test`: Run tests (currently not implemented).
- `bun check`: Run the linting process.
