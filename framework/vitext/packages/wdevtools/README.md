# wdevtools

## Introduction

`wdevtools` is a Nuxt-based application that provides a powerful set of development tools for your Vite projects. It's designed to be a central hub for monitoring, debugging, and interacting with your application during development.

## Features

- **Component Inspector**: View and interact with your component tree.
- **State Management**: Monitor your Pinia stores in real-time.
- **Asset Viewer**: Inspect your static assets.
- **API Routes**: Test and debug your Nitro server routes.
- **Dark Mode**: Comes with a beautiful dark mode out of the box.

## Installation

| npm | pnpm | yarn | bun |
| --- | --- | --- | --- |
| `npm install wdevtools -D` | `pnpm add wdevtools -D` | `yarn add wdevtools -D` | `bun add wdevtools -D` |

## Usage

`wdevtools` is intended to be run as a separate application alongside your main Vite project.

1.  **Clone the repository**
2.  **Install dependencies**: `bun install`
3.  **Run the dev server**: `bun run dev`
