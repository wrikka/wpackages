# @wai/rules

Effect-based utilities for loading, validating, and working with rules (frontmatter-driven conventions).

## Tech Stack

- Bun
- TypeScript
- Effect (Effect TS)

## Install

From the monorepo root:

```bash
bun install
```

## Development

```bash
bun --cwd bun-packages/rules run dev
```

## Testing

```bash
bun --cwd bun-packages/rules run test
```

## Lint / Typecheck

```bash
bun --cwd bun-packages/rules run lint
bun --cwd bun-packages/rules run typecheck
```

## Build

```bash
bun --cwd bun-packages/rules run build
```

## Verify

```bash
bun --cwd bun-packages/rules run verify
```
