# @wai/memories

Effect-based utilities for memory storage, retrieval, and related workflows.

## Tech Stack

- Bun
- TypeScript
- Effect (Effect TS)

## Install

From the monorepo root:

```bash
bun install
```

## Development

```bash
bun --cwd bun-packages/memories run dev
```

## Testing

```bash
bun --cwd bun-packages/memories run test
```

## Lint / Typecheck

```bash
bun --cwd bun-packages/memories run lint
bun --cwd bun-packages/memories run typecheck
```

## Build

```bash
bun --cwd bun-packages/memories run build
```

## Verify

```bash
bun --cwd bun-packages/memories run verify
```
