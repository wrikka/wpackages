# @wpackages/math

Math utilities package for wpackages monorepo.

## Installation

```bash
bun add @wpackages/math
```

## Usage

```typescript
import { add, subtract, multiply, divide } from '@wpackages/math'

console.log(add(1, 2)) // 3
console.log(subtract(5, 3)) // 2
```

## Features

- Basic arithmetic operations
- Advanced mathematical functions
- Type-safe utilities
- Tree-shakeable exports
