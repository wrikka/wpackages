# ast-grep-rules

Package for ast-grep-rules

Vite plugin for ast-grep rules integration

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```

## Development

```bash
bun run build
bun run test
```

## Available Scripts

- `build`: tsdown
- `dev`: tsdown --watch
- `test`: vitest
- `lint`: tsc --noEmit && oxlint --fix --type-aware
- `typecheck`: tsc --noEmit
- `watch`: bun --watch src/index.ts
- `format`: dprint fmt
