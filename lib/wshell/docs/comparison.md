# wshell vs. Nushell Comparison

| Feature                  | wshell (Anticipated)                              | Nushell                                                  |
| ------------------------ | ------------------------------------------------- | -------------------------------------------------------- |
| **Data Model**           | Structured (Effect-TS Schemas)                    | Structured (Tables, Lists, Primitives)                   |
| **Language Paradigm**    | Functional, Type-Safe                             | Functional, with imperative features                     |
| **Type System**          | Strong, static typing via TypeScript & Effect-TS  | Gradually typed, strong type system at runtime           |
| **Ecosystem**            | Node.js/Bun ecosystem, Effect-TS libraries        | Custom plugins (Rust), scripts                           |
| **Platform Support**     | Anywhere Bun/Node.js runs (Linux, macOS, Windows) | Linux, macOS, BSD, Windows                               |
| **Developer Experience** | Excellent type inference, composable programs     | Rich data visualization, smart completions, clear errors |
