# api-builder

Package for api-builder

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```

## Development

```bash
bun run build
bun run test
```

## Available Scripts

- `dev`: bun run examples/app.ts
- `build`: bun build src/index.ts --outdir dist
- `lint`: tsc --noEmit && oxlint --fix --type-aware
- `test`: bun test
- `watch`: bun --watch src/index.ts
- `format`: dprint fmt
