# @wpackages/unused

Find unused files, dependencies and exports in your JavaScript/TypeScript projects.

## Comparison with Knip

While `@wpackages/unused` offers a basic check for unused dependencies, `Knip` is a more powerful and comprehensive tool for project-wide analysis.

| Feature                  | @wpackages/unused | Knip      |
| ------------------------ | :---------------: | :-------: |
| **Unused Dependencies**  |         ✔️         |     ✔️     |
| **Unused Files**         |         ❌         |     ✔️     |
| **Unused Exports**       |         ❌         |     ✔️     |
| **Configuration**        |         ❌         |     ✔️     |
| **Plugins/Extensibility**|         ❌         |     ✔️     |

For a more detailed breakdown, see the [full comparison document](./docs/comparison.md).
