# error

Package for error

Error handling utilities and custom error types

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```

## Development

```bash
bun run build
bun run test
```

## Available Scripts

- `lint`: tsc --noEmit && oxlint --fix --type-aware
- `build`: tsdown src/index.ts --out-dir dist
- `test`: vitest run
- `watch`: bun --watch src/index.ts
- `dev`: bun run src/index.ts
- `format`: dprint fmt
