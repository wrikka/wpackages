# store

Package for store

A tiny, framework-agnostic state management library inspired by Nanostore.

## Installation

```bash
bun install
```

## Usage

```bash
bun run dev
```

## Development

```bash
bun run build
bun run test
```

## Available Scripts

- `build`: tsc
- `test`: vitest
- `lint`: tsc --noEmit && oxlint --fix --type-aware
- `watch`: bun --watch src/index.ts
- `dev`: bun run src/index.ts
- `format`: dprint fmt
