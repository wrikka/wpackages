# TUI Command Palette

A terminal UI command palette similar to fzf in nvim, built with Rust.

## Features

- Interactive search/filter
- Keyboard navigation (up/down arrows)
- Select items with Enter
- Exit with Esc

## Dependencies

- ratatui - TUI framework
- crossterm - terminal handling
- anyhow - error handling

## Usage

`ash
cargo run
`

## Key Bindings

- Type - Filter items
- Up/Down - Navigate items
- Enter - Select item
- Esc - Exit
